# goimgext
Image Extension for Go

# How to install
<pre>
go get github.com/shamsher31/goimgext
</pre>

# How to use
<pre>
package main

import (
	"fmt"
	"github.com/shamsher31/goimgext"
)

func main() {
	fmt.Println(imgext.Get())
}
</pre>

# Why
This package is inspired by [arthurvr](https://github.com/arthurvr/image-extensions)
